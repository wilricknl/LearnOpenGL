# Model

### Commands

| Key      | Description               |
|----------|---------------------------|
| Escape   | Exit program.             |
| W        | Move camera forward.      |
| S        | Move camera backward.     |
| D        | Move camera to the right. |
| A        | Move camera to the left.  |
| Q        | Move camera upward.       |
| E        | Move camera downward.     |
| Scroll   | Increase/decrease fov.    |
| Mouse    | Look around               |
